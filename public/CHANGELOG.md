# Changelog

Здесь по версиям собрано, **что изменилось** в программе GoldSrc Config Engineer — коротко и по делу.

---

## [3.2.0] — 2026-04-06

### Главная и готовность

- Под блоком **«Готовность окружения»** — **заполненность конфига**: процент и прогресс по **10 разделам** (параметры, бинды, режим, сеть, видео, прицел, мышь, звук, FPS, закупка).
- **Палитра команд (⌘K / Ctrl+K):** можно искать по словам про готовность и заполненность, чтобы перейти на главную.

### Тема и оформление

- При **первом запуске** тема берётся **из системы** (светлая или тёмная), пока вы сами не выберете в настройках.
- Единая сетка страниц, подписи и отступы; аккуратнее карточки, сайдбар, заставка с **номером версии**, кнопки успешных действий и ползунки.

### Навигация

- Дополнительные **сочетания клавиш** для быстрых переходов (настройки, главная, диагностика, профили, мастер, режимы, сравнение, пресеты, прицел, чувствительность) — полный список в **справке (F1)**.
- **Контекстное меню** по странице и палитра обновлены под эти разделы.

### Данные и удобство

- В **настройках** и **диагностике** видны пути к данным приложения и базе; можно **копировать сводку** проверок и ответ **«О приложении»**.
- **Профили**, **история снимков**, **импорт по ссылке** и остальное — см. предыдущие версии и подробный журнал для разработчиков в репозитории.

## [3.1.3] — 2026-04-04

### Новое

- Страница **«Чувствительность мыши»**: наглядно сантиметры на оборот и связь DPI с чувствительностью; есть в меню, поиске и быстрых действиях.
- **Совет дня** на главной — меняется в зависимости от даты.
- **Сравнение** — можно скопировать результат или сохранить в текстовый файл.
- **Импорт** — ограничение размера вставки с понятным сообщением.
- **Режимы и про-пресеты** — сортировка «как в списке» или по алфавиту.
- **Просмотр** — «Копировать всё» и перенос очень длинных строк.
- В поиске по командам — сниппет для чувствительности; группа команд переименована в **«Данные»**.

### Тексты

- Подписи в интерфейсе сформулированы нейтрально, без упоминаний внутренних технологий.
- В журнале событий разделы подписываются понятно, без служебных кодов.

## [3.1.1] — 2026-04-04

### Инструменты

- **Параметры запуска** — собрать строку для Steam, скопировать одной кнопкой; есть в меню, поиске, на главной и с экспорта; горячая клавиша **Ctrl+L** (на Mac — **⌘L**).
- **Поиск по программе** — подсказка про параметры запуска внизу окна.
- **Режимы и пресеты** — поиск по названию и содержимому.
- **Прицел** — подсказки к настройкам и динамическому прицелу.
- **Импорт** — вставка текста из буфера обмена.
- **Сравнение** — поменять местами два конфига без повторного выбора файлов.

### Настройки и журнал

- Подробный журнал можно включить в настройках; просмотр и очистка — на странице **Диагностика**.
- Горячие клавиши быстрых переходов не мешают набору текста в полях (кроме открытия поиска **K**).

## [3.1.0] — 2026-04-03

### Главная

- Стартовый экран с быстрыми действиями, списком недавних конфигов и генерацией с выбором режима; запоминается последний выбранный режим.
- Список недавних обновляется после импорта, мастера, экспорта и работы с режимами и пресетами.

### Навигация

- Боковое меню с группами разделов и сворачиванием; тёмная тема для панели приведена к единому виду.
- Упрощены «хлебные крошки» на главной.
- **Ctrl+S** (на Mac — **⌘S**) ведёт к экспорту; в поиске по программе — очистка недавних и подсказка про сохранение.
- Справка по **F1** и настройки используют один список горячих клавиш.

## [2.4.0] — 2026-04-02

### Скорость и удобство

- Тяжёлые экраны (прицел, сравнение, просмотр и др.) открываются с заметкой «загрузка», а не пустым окном.
- В **сравнении** сначала появляется диаграмма, строки таблицы подгружаются по частям — интерфейс не подвисает.
- При наведении на пункт меню данные для страницы могут подгружаться заранее — переход ощущается быстрее.

### Память и стабильность

- При переходе между страницами освобождается часть ресурсов; история конфигов ограничена по размеру.
- Профили можно хранить в сжатом виде — меньше места на диске.
- Иконки и служебные объекты кэшируются аккуратнее; для отладки есть дополнительная информация по желанию.

## [2.1.0] — 2026-04-02

### Новые возможности

- **Шаблоны конфигов** — готовые старты для новичка, соревновательной игры, казуала, стрима и LAN.
- **Избранные команды** — список любимых CVAR для быстрого доступа.
- **Импорт перетаскиванием** файла в окно (где поддерживается системой).
- **Пакетный экспорт** — сохранить конфиги всех профилей в папку разом.

### Прицел

- Рядом показать **два прицела** — ваш и пресет.
- Сохранить картинку прицела в файл.
- Опционально **силуэт цели** для наглядности.

### Прочее

- После экспорта можно сразу открыть диагностику; сравнение и история — доработаны для удобного отчёта.
