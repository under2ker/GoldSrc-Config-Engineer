# 🎯 ТЕХНИЧЕСКОЕ ЗАДАНИЕ: ВНЕШНИЙ ВИД (UI/UX)

## 1. Общая концепция дизайна

### 1.1 Дизайн-язык

* Стиль: **Modern Desktop Pro Tool (DevTool-style UI)**
* Основа: **dark-first UI**
* Вдохновение:

  * VS Code
  * Figma
  * Linear
  * Raycast

### 1.2 Визуальная модель

* Основа: **layered interface**

  * Base (фон)
  * Surface (карточки)
  * Overlay (модалки, popover)
* Использовать:

  * **Glassmorphism (ограниченно)** для overlay (header `backdrop-blur`, модалки)  * Не использовать повсеместно (только:

    * modals
    * command palette
    * tooltips)

---

## 2. Цветовая система (Design Tokens)

### 2.1 Основные цвета

```css
--bg-primary: #0b0f14;
--bg-secondary: #11161c;
--bg-tertiary: #161b22;

--surface: #1c2128;
--surface-hover: #242a33;

--border: #2a2f36;
--border-soft: rgba(255,255,255,0.05);

--text-primary: #e6edf3;
--text-secondary: #9da7b3;
--text-muted: #6e7681;

--accent: #3b82f6;
--accent-hover: #2563eb;
--accent-soft: rgba(59,130,246,0.15);

--danger: #ef4444;
--success: #22c55e;
--warning: #f59e0b;
```

### 2.2 Требования

* Контраст WCAG ≥ 4.5
* Accent не должен «жечь глаза»
* Hover всегда отличается от base

---

## 3. Типографика

### 3.1 Шрифты

* UI: `Inter / Segoe UI`
* Code: `JetBrains Mono / Consolas`

### 3.2 Иерархия

| Тип   | Размер | Вес |
| ----- | ------ | --- |
| H1    | 24px   | 600 |
| H2    | 20px   | 600 |
| H3    | 16px   | 500 |
| Body  | 14px   | 400 |
| Small | 12px   | 400 |

---

## 4. Layout система

### 4.1 Общая сетка

* Sidebar: **240px (collapsed: 64px)**
* Content: flex-grow
* Padding:

  * page: 24px
  * cards: 16px

### 4.2 Структура

```
Titlebar
Sidebar | Main Content
        | Header
        | Page Content
```

### 4.3 Поведение

* Sidebar:

  * collapsible (persist в store)
* Header:

  * sticky
* Content:

  * scrollable

---

## 5. Компоненты UI

### 5.1 Button

Состояния:

* default
* hover
* active
* disabled
* loading

Требования:

* loading → spinner + disable
* минимальная ширина: 80px

---

### 5.2 Card

* border-radius: 12px
* padding: 16px
* hover:

  * slight elevation
  * border highlight

---

### 5.3 Input / Slider / Select

* unified height: 36px
* focus:

  * accent outline
* error state:

  * red border

---

### 5.4 Dialog / Modal

* glass effect (blur 10px)
* backdrop:

  * rgba(0,0,0,0.5)

---

### 5.5 Context Menu (обязательно)

(из backlog / идей продукта)

Добавить:

* CVAR reset
* copy/paste
* preset actions

---

### 5.6 Breadcrumbs

(из навигационных паттернов)

Требования:

* всегда под заголовком страницы
* clickable

---

## 6. Навигация

### 6.1 Sidebar

* иконки + текст
* active:

  * accent background
* hover:

  * subtle highlight

### 6.2 Command Palette (Cmd+K)

* обязательно:

  * поиск страниц
  * поиск CVAR
  * быстрые действия

---

## 7. Анимации

### 7.1 Общие правила

* использовать **Framer Motion**
* длительность: 120–200ms

### 7.2 Типы

* page transition: fade + slight slide
* hover: scale 1.02
* modal: scale + fade

---

## 8. Производительность UI

Обязательно (критично):

* virtualization (TanStack Table)
* debounce (inputs, sliders)
* lazy loading страниц / чанков (частично: ленивый `diff`)
* skeleton screens (`src/components/ui/skeleton.tsx`)

---

## 9. Специальные UI-фичи

### 9.1 Particle background (опционально)

(опционально из идей)

Ограничения:

* opacity ≤ 0.2
* не влиять на FPS

---

### 9.2 Crosshair Canvas

* 60 FPS
* debounce redraw
* no re-render React

---

### 9.3 Diff Viewer

* цвета:

  * removed: red
  * added: green
* построчная подсветка

---

## 10. UX требования

### 10.1 Время отклика

* UI action: < 100ms
* генерация: < 300ms

### 10.2 Feedback

* всегда:

  * loading
  * success toast
  * error toast

---

### 10.3 Ошибки

* не ломают UI
* показываются пользователю

### Навигация

* [~] Command palette (Cmd+K) — разделы + CVAR + **обновление каталога** + справка + **очистка недавних** `.cfg`; расширенные keywords; полный fuzzy — позже
* [x] Sidebar active states
* [x] Deep linking (URL routes)

---

### Анимации

* [~] Page transitions — CSS `.page-enter` вместо Framer Motion

* [~] Modal — без отдельной motion-библиотеки

* [x] Skeleton screens — режимы/пресеты + обновлённый **`PageLoading`** под сетку карточек
* [x] Memoization — сравнение: **bar + radar** данные через `useMemo`

### UX

* [~] Loading states — **экспорт** и **быстрая настройка**: скелетон до загрузки каталога; импорт: **EmptyState**; кнопки `loading` по страницам
